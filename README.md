# Do_chat_luong_khong_khi
Dùng để đo bụi và các loại khí khác và tự động hóa việt mở và đẩy quạt ra

## Features
- Detect air quality using gas sensor (MQ series)
- Display system status using LEDs
- Automatically control fan via relay
- Real-time monitoring and response

## Hardware Components
- Arduino Uno
- Gas Sensor (MQ-2 / MQ-135)
- Relay Module
- DC Fan
- LEDs (status indicators)
- Breadboard & jumper wires
- Power supply (battery)

## Working Principle
The gas sensor continuously measures air quality. 
The microcontroller processes the sensor data and:
- Activates the fan when air quality is poor
- Turns on warning LEDs
- Keeps the fan off when air is clean
